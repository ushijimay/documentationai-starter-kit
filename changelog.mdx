## Migration Guide

Follow these steps to update your Skill3 instance to the latest version.

<Callout kind="alert">
Before migrating, back up your database and configuration files.
</Callout>

<Steps>
  <Step title="Backup Data" icon="database">

    Create a full backup of your current setup.

````bash
pg_dump -U youruser skill3_db > skill3_backup_$(date +%Y%m%d).sql
````

  </Step>
  <Step title="Update Package" icon="package">

    Install the latest version using your package manager.

    <CodeGroup tabs="npm,yarn">
````bash
npm install skill3@latest
````
````bash
yarn add skill3@latest
````
    </CodeGroup>

  </Step>
  <Step title="Run Migrations" icon="git-merge">

    Execute database migrations.

````bash
npx skill3 migrate
````

  </Step>
  <Step title="Restart Service" icon="refresh-cw">

    Restart your Skill3 application.

````bash
systemctl restart skill3
# or
pm2 restart skill3
````

  </Step>
</Steps>

<Expandable title="Detailed Migration Notes for v1.2.0" default-open="false">

If you encounter issues with custom categories, update your configuration:

````json
{
  "categories": {
    "enabled": true,
    "defaults": ["business", "certification", "conversation", "travel"]
  }
}
````

Review the full migration script in the release notes on GitHub.

</Expandable>
